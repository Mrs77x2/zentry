# Visión de Zentry

Zentry busca ser una plataforma integral para la gestión de pequeñas y medianas empresas (PYMEs).

## Problema
Muchas empresas no cuentan con software accesible y flexible para gestionar su operación.

## Solución
Un sistema modular que permita adaptarse a distintos tipos de negocio.

## Tipos de usuarios

### Usuario público
- Consulta catálogo
- No requiere autenticación

### Cliente
- Puede interactuar con servicios o compras

### Empleado
- Opera dentro del sistema

### Administrador
- Control total del sistema
