# Arquitectura de Zentry

## Tipo de arquitectura
Monolito modular

## Estructura

backend/
 ├── auth/
 ├── user/
 ├── inventory/
 ├── sales/
 ├── client/
 └── common/

## Patrones utilizados
- Layered Architecture
- DTO Pattern
- Dependency Injection

## Objetivo
Mantener un sistema escalable y preparado para migrar a microservicios en el futuro.
