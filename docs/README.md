# Zentry

Zentry es una plataforma SaaS modular para la gestión de PYMEs.

## 🎯 Objetivo
Centralizar la operación de negocios como:
- Inventario
- Empleados
- Clientes
- Ventas
- Finanzas

## 🧠 Enfoque
Sistema modular, escalable y adaptable a distintos tipos de negocio:
- Gimnasios
- Escuelas
- Autolavados
- Empresas de servicios
- Venta de productos

## 🛠 Tecnologías
- Java (Spring Boot)
- PostgreSQL
- Docker
- AWS
- HTML, CSS, JavaScript

## 🔐 Seguridad
- Autenticación con JWT
- Roles y permisos (RBAC)

## 🚧 Estado del proyecto
Fase inicial: diseño de arquitectura y módulo de autenticación

Proyecto desarrollado como base profesional para software empresarial.
